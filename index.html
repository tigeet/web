The Markdown Basics

A minimalistic and readable markdown language

"Simplicity is the ultimate sophistication" Leonardo da Vinci

Syntax

Headings

HTML headings are titles or subtitles

# Heading level 1
## Heading level 2

Blockquotes

Blockquote indicates that the enclosed text is an extended quotation

> This is a blockquote

Lists

List allows to organize items into ordered and unordered lists

1. First item
2. Second item
3. Third item
  1. Indented item
  2. Indented item
4. Fourth item
